# Password Strength Checker

A Password Strength Checker is a program that looks at a password and tells the user how strong the password is. Here is a simple project for people who're new, to Python that checks how strong a password is based on the password strength. This Password Strength Checker project evaluates the password strength of a password based on:
- length
- uppercase letters
- lowercase letters
- numbers

It gives a rating: Weak / Medium / Strong.
